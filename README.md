Clipo
核心功能
自动记录剪贴板历史，支持文本、代码、链接、图片、文件等多种内容类型。
卡片式快捷面板，支持快速浏览、翻页、复制、删除和选择记录。
完整历史窗口，支持搜索、筛选和管理大量剪贴板内容。
支持多选模式，可批量复制、删除或发送到 AI。
支持重要内容置顶，方便长期保留常用记录。
支持相似内容查找，快速找回相关剪贴板记录。
新增 AI 搜索，可用自然语言查找历史内容。
新增 AI 模式，可对选中内容进行总结、翻译、改写和整理。
支持图片 AI 索引，让图片内容也可以被搜索。
支持自定义 AI 接口、模型和 API Key。
支持鼠标中键快速呼出面板并悬停选择卡片。
提供浅色 / 深色模式、主题色和液态玻璃风格界面。
